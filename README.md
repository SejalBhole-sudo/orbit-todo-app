# Orbit To-Do App

Orbit To-Do is a modern task management web application built with React.  
It helps users organize daily tasks with a clean pastel interface and simple productivity features.

Tasks persist using browser localStorage so they remain available even after refreshing the page. Completed tasks automatically disappear the next day to encourage a fresh start each day.

---

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Separate active and completed task lists
- Circular toggle checkbox
- Daily reset of completed tasks
- Task counter for remaining tasks
- Persistent storage using localStorage
- Responsive pastel UI design
- Empty state message when no tasks exist

---

## Tech Stack

- React (Vite)
- JavaScript
- CSS
- localStorage
